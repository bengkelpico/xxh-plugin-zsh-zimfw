## Install
Install from any Github repo:
```
XXH +I https://github.com/xxh/xxh-plugin-zsh-zimfw
```
Connect:
``` 
xxh yourhost +s zsh +if
```

## Seamless mode
To use seamless mode with saving your theme and plugins use `source` command: 
```
source xxh.zsh myhost
```
